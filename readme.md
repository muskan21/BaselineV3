#Project Description
Baseline Problems generates receipts for the given set of items including imported items, after applying the applicable tax rules.

#Environment Setup
1. Any OS.
2. JDK(1.7 or above)
3. Apache Ant

#Build The CodeBase
Run The Following Command :
`ant build`

#Run The Project
###Run The Testcases
Execute The Following Command :
`ant runTest`
###Run The Codebase
Execute The Following Command :
`ant run`
